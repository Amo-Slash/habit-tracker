# Habit Tracker
A simple habit tracking web app i built for my Melsoft Academy JavaScript project.

## What it does

- Add a daily habit with a name, a target (how many days per week) and a category
- Validates the form before adding — shows an error message if something is wrong
- Displays all habits in a list with a checkbox, streak counter and delete button
- Tick the checkbox to mark a habit done for today, streak goes up by 1
- Untick it and the streak goes back down
- Delete a habit with the x button
- Summary at the top shows total habits, how many are done today and the completion percentage

## How to run it

No installs needed. Just download the files and open index.html in your browser. Thats it.

If you have VS Code you can right click index.html and open with Live Server.

## Files

- index.html — the structure of the page
- style.css — all the styling
- script.js — all the logic

## Built with

- HTML
- CSS (flexbox, media queries)
- Vanilla JavaScript (DOM manipulation, event listeners, array methods)

## What i learned building this

- How to use classes and functions to organize validation logic
- How to build and update the DOM using createElement and appendChild
- How to use addEventListener instead of inline onclick
- How flexbox works for laying out cards side by side
- How media queries make the layout responsive on smaller screens
